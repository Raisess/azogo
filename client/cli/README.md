# BitFunky CLI

A simple cli written in C++, using [cli-creator](https://github.com/Raisess/cli-creator).

### Installing:

- Only inside the cli folder root!

```shell
make
make install
```

### How to use:

```shell
bitfunky --torrent=./path/file.torrent
bitfunky --torrent=./path/file.torrent,./path/output
```

### Uninstalling:

- Only inside the cli folder root!

```shell
make uninstall
```

that is all.
